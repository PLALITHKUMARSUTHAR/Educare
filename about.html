Description about website

This website, hosted on GitHub, was built using basic HTML, CSS, JavaScript, Bootstrap, and custom snippets. HTML was used for structuring content, while CSS provided styling for a responsive layout. JavaScript added interactive elements, enhancing user engagement. Bootstrap facilitated rapid development with its grid system and pre-styled components. Custom snippets were employed to add specific features and optimize functionality. Hosted on GitHub Pages, the site benefits from easy deployment and version control through Git, ensuring seamless updates and collaborative development.
